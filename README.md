# ABC Company Employee Data Analysis

## Project Overview
This project provides a comprehensive analysis of employee data for ABC Company. Using a dataset of 458 employees, we performed data preprocessing, exploratory data analysis (EDA), and graphical visualization to uncover workforce trends.

## Preprocessing Steps
*   **Data Integrity**: Addressed inconsistencies in the 'Height' column by replacing values with randomized integers between 150 and 180 to ensure a uniform distribution for analysis.
*   **Data Cleaning**: Verified data types and ensured there were no missing values across the 9 columns.

## Key Analysis Tasks
1.  **Team Distribution**: Calculated the headcount and percentage split per team.
2.  **Position Segregation**: Classified employees based on roles (e.g., Forward, Guard, Center).
3.  **Age Demographics**: Identified the most common age group within the company.
4.  **Salary Expenditure**: Determined which teams and positions consume the largest portion of the budget.
5.  **Age-Salary Correlation**: Investigated the statistical relationship between employee age and their compensation.

## Insights (Data Story)
*   **Workforce Youth**: The majority of employees fall within the 25-30 age range, suggesting a focus on developing young talent.
*   **Salary Drivers**: While age shows a slight positive correlation with salary, the most significant expenditure is often tied to specialized positions like 'Center'.
*   **Team Concentration**: Certain teams have a higher percentage of the total workforce, indicating central hubs of operations.

## Technologies Used
*   **Python**: Primary programming language.
*   **Pandas**: For data manipulation and aggregation.
*   **Matplotlib & Seaborn**: For creating professional visualizations.
*   **Jupyter Notebook**: For interactive code execution and documentation.
